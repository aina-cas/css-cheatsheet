CSS Cheatsheet

Selectors
div all div tags
div, span all div tags and all span tags
div span all span tags inside div
#content element with ID "content"
.box all elemenst with class "box"
* unversal, selects all elements
#box* all elements inside #box
a:link,
a:active,
a:visited		 links in normal state, in clicked state, and visited state
a:hover link with mouse over it
div > span all span's one-level deep in a div
Box Model
Text
font-family Font used, e.g. Helvetica, Arial
font-size Text size, e.g. 50px, 3em
color Text color, #FFF
font-weight How bold the text appears
font-style Style of font, e.g. italic
text-declaration Sets a variety of effects on text, e.g. underline, overline, none
text-align Alligns text, e.g. center, right
line-height Spacing between lines
letter-spacing Spacing between letters
text-indent Indents the first line
text-transform Applies formating to text, e.g. uppcase, lowercase
vertical-align Aligns relative to baseline
Border, Padding & Margin
border Sets border style for all borders in the format: border:(solid, dashed, dotted, double) (width) (color), e.g. border:solid 1px #000
border-top
border-bottom
border-left
border-right		 Sets border style for a specific border (same property syntax used for padding and margin, e.g. margin-left

Syntax
selector {property:value;} Write styles for any element
link rel="stylesheet" href="./example.css>" External Style Sheet
style >
selector{property:value;}
/style>		 Internal Style Sheet
tag style="property:value"> Inline Style
Positioning
position Places elements on screen, e.g. absolute, fixed, relative
float Stacks elements horizontally in a particular direction, e.g. left
top, right, bottom, laft Specifies the offset used in absolute, fixed and relative positions
display Sets how element is displayed in the doc flow, e.g. inline, block, none
z-index Set the stacking order of elements, e.g. z-index of 1 is bellow 2
overflow Sets what happens to content outside of container, e.g. hidden, scroll
clear Specifies how elements behave, when they bump into each other
Pseudo Selectors
:hover triggered when the user hovers over an element with the curso
:active represents an element (such as a button) that is being activated by the user
:focus Represents an element (such as a form input) that has received focus
:link Represents an element that has not yet been visited
:visited Represents links that the user has already visited
:first-line Aplies styles to the first line of a block-level element
:first-letter Applies styles to the first letter of the first line of a block-level element
